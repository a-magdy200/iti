# Store Rest API using native PHP

```
Base URI: **index.php/items**
```

**GET**

Returns all items

`/`

**POST**

Creates a new item

`/`

**GET**

Returns single item

`/{id}`

**PUT/PATCH**

Updates single item

`Header | Content-Type: application/x-www-form-urlencoded`

`/{id}`

**DELETE**

Deletes single item

`/{id}`
